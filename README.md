## API Endpoints

- **POST /api/cities**: Create a new city.
- **GET /api/cities**: Retrieve all cities.
- **GET /api/cities/:id**: Retrieve a city by ID.
- **PUT /api/cities/:id**: Update a city by ID.
- **DELETE /api/cities/:id**: Delete a city by ID.
